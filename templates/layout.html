About the Cancer Data

The data used in this web app is from the CORGIS Dataset Project and can be found here. The data contains statistics about cancer death rates by state or by 100,000 population from the years 2007 to 2013 so the data is a bit outdated but still valubale. If you would like to you can do your own research on the (National Cancer Institute) NCI's offical website were they have a lot of statistics for death rates, survival rates, cancer's biolodgy and much much more you can check them out here
